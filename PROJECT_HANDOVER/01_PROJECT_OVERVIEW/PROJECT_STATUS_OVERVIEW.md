## 系統架構概覽

### 主要模組
```
src/
├── dcf_calculator/     # DCF 估值核心
├── data_fetcher/       # 數據抓取模組
├── trading_system/     # 交易系統
├── ui/                 # 使用者介面
└── utils/             # 工具函數
```

### 核心依賴
- **FinMind**: 財務數據來源
- **Streamlit**: Web 介面框架  
- **pandas/numpy**: 數據處理
- **requests**: API 通訊
- **lxml**: XML/HTML 解析

## 數據流程
1. **數據抓取**: FinMind API + TWSE 網站
2. **數據處理**: 財務報表標準化
3. **DCF 計算**: 多情境估值分析
4. **結果展示**: Streamlit 互動介面
5. **快取機制**: 本地儲存減少 API 調用
