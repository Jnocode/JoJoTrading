```bash

python -m streamlit run app.py --server.port 8503
```

1. **選擇預設配置**: 在側邊欄選擇成長股或 DCF 預設

3. **管理配置**: 載入、刪除、匯出/匯入配置檔案
4. **分享配置**: 使用 JSON 格式匯出配置與他人分享
