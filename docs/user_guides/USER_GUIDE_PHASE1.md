## 🎛️ Phase 1 新功能使用說明

### 增強功能控制面板 (側邊欄)

1. **啟用增強型 DCF**
   - 切換按鈕控制是否使用 Phase 1 優化功能
   - 開啟時使用智能集成處理器
   - 關閉時使用原始 DCF 計算

2. **數據質量閾值調整**
   - 滑桿範圍: 0.5 - 0.9
   - 建議設定: 0.7 (平衡準確性與可用性)
   - 高於閾值: 使用增強型 DCF
   - 低於閾值: 自動備援至標準 DCF

3. **情景分析設定**
   - 啟用後會計算樂觀/基準/悲觀三種情景
   - 提供更全面的估值範圍
   - 幫助評估不確定性風險

4. **蒙地卡羅模擬**
   - 模擬次數: 100-5000 (預設 1000)
   - 波動率: 0.05-0.3 (預設 0.1)
   - 生成估值分布與風險指標

5. **異常檢測**
   - 敏感度: 0.1-0.5 (預設 0.2)
   - 自動識別數據異常
   - 提高估值可靠性

## 📊 新增顯示內容

### 數據質量指標
- **質量評分**: 0-1 分數顯示數據可靠度
- **品質狀態**: 高品質/中品質/低品質標示
- **使用方法**: 顯示當前採用的 DCF 方法

### 增強型估值結果
- **情景分析**: 三種情景下的估值結果
- **蒙地卡羅**: 平均值、標準差、VaR 風險指標
- **敏感性**: 關鍵參數變化影響分析

### 風險評估
- **數據風險**: 基於質量評分的可靠度
- **模型風險**: 估值方法的適用性評估
- **市場風險**: 基於波動率的風險量化

## 🔍 使用建議

### 高品質股票分析
1. 選擇財務數據穩定的大型股
2. 啟用所有 Phase 1 增強功能
3. 使用較低的質量閾值 (0.6-0.7)
4. 參考情景分析與蒙地卡羅結果

### 中小型股票分析
1. 提高質量閾值至 0.8
2. 關注數據質量評分
3. 如數據品質不佳，信賴標準 DCF 結果
4. 參考異常檢測提示

### 成長股分析
1. 適當提高波動率設定 (0.15-0.2)
2. 重視情景分析的樂觀情景
3. 關注敏感性分析結果
4. 考慮更長期的成長預期

## 🛠️ 疑難排解

### 常見問題

1. **增強功能無法啟用**
   - 檢查 `USE_ENHANCED_DCF` 是否為 True
   - 重新啟動應用程式
   - 檢查依賴套件是否完整安裝

2. **數據質量評分過低**
   - 檢查財務數據完整性
   - 嘗試選擇不同的時間範圍
   - 考慮使用標準 DCF 方法

3. **蒙地卡羅模擬過慢**
   - 減少模擬次數 (500-1000)
   - 檢查系統資源使用狀況
   - 考慮在高性能電腦上運行

4. **估值結果異常**
   - 檢查輸入參數合理性
   - 驗證財務數據正確性
   - 參考異常檢測提示

### 效能優化

1. **提升運算速度**
   - 使用 SSD 硬碟
   - 增加系統記憶體
   - 關閉不必要的背景程式

2. **記憶體使用優化**
   - 減少同時分析的股票數量
   - 定期清理瀏覽器快取
   - 重新啟動應用程式
