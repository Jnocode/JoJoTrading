Why this matters

這個 demo 是求職展示版：它證明我能把資料處理、交易規則、回測驗證、CI/CD 與 AI Agent 審計整合成一個可看、可跑、可維護的工程作品。完整交易憑證與 live broker API 不會放在公開 demo。
