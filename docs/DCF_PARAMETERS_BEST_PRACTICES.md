## ⚠️ 重要注意事項

1. **參數不是固定的**: 應根據市場環境、個股特性動態調整
2. **保守原則**: 寧可低估也不要高估成長率
3. **安全邊際**: 篩選閾值是保護投資者的重要機制
4. **定期檢視**: 建議每季檢視並調整參數設定
