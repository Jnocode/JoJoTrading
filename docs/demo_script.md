## 🎬 錄製技巧

### 📹 錄製前準備
1. **清理畫面**
   - 關閉不必要的瀏覽器分頁
   - 隱藏桌面圖示
   - 設定瀏覽器為全螢幕（F11）

2. **Demo 數據準備**
   - 預先載入台積電（2330）數據
   - 準備好模擬結果（避免現場等待）
   - 測試所有功能是否正常

3. **錄製設定**
   - 解析度：1920x1080
   - FPS：30
   - 麥克風測試（清晰度）

### 🎙️ 旁白技巧
1. **語速控制**
   - 不要太快，保持節奏
   - 重要數據時稍微放慢

2. **語氣自然**
   - 不要太僵硬，像在介紹專案給朋友
   - 可以適度加入感情

3. **發音清晰**
   - 技術名詞要念對（AsyncIO、Monte Carlo）
   - 數字要清楚（60%、75%）

### 📝 字幕製作
1. **雙語字幕**
   - 上方：英文
   - 下方：中文
   - 字體：微軟正黑體 / Arial

2. **重點數據標註**
   - 用顏色強調（黃色/綠色）
   - 數據出現時暫停 1-2 秒

3. **時間軸同步**
   - 字幕與旁白完全同步
   - 重要數據可以提前 0.5 秒顯示
