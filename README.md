> **🚀 Phase 1 優化版本現已發布！** 
> 
> 全新的數據質量驗證系統與增強型 DCF 模型，為投資決策提供更精確、可靠的分析工具。

## 🎯 核心功能

### 基礎功能
- **🌐 多語介面**：支援中文/英文切換，所有自訂UI皆可即時切換語言
- **🔍 產業/個股篩選**：可依產業或自選個股進行財報與估值分析
- **📈 成長股過濾**：可依營收/盈餘CAGR、ROE、毛利率等指標自訂過濾成長股
- **💰 DCF估值**：支援多階段成長率、折現率自動/手動設定，並可敏感度分析
- **⚙️ 參數自動化**：無風險利率、通膨率、永續成長率可自動API獲取或手動輸入，來源明確
- **🔧 除錯模式**：模組化debug_tools，支援清除快取、API延遲調整、模擬數據測試
- **📊 結果匯出**：支援CSV/Excel下載，所有參數與來源可記錄於報表

### 🆕 Phase 1 優化功能
- **📋 數據質量驗證**：多維度數據品質評估與異常檢測
- **🎯 增強型 DCF 模型**：CAPM 動態折現率與情景分析
- **🎲 蒙地卡羅模擬**：風險量化分析與 VaR 評估
- **🎛️ 智能集成處理**：基於數據質量的自動方法選擇
- **📊 情景分析**：樂觀/基準/悲觀三種情景並行計算
- **⚡ 即時參數調整**：UI 控制面板支援動態配置

## 🏗️ 系統架構

### 核心模組
- `app.py`：Streamlit主介面，負責UI、語言切換、參數傳遞
- `jojo_state_machine.py`：狀態機流程控制，負責資料流與狀態轉換
- `data_handler.py`：資料抓取、財報解析、估值計算與 Phase 1 模組集成
- `data_fetching.py`：FinMind API 資料抓取與處理

### Phase 1 優化模組
- `modules/data_validator.py`：數據質量驗證系統
- `modules/enhanced_dcf.py`：增強型 DCF 計算引擎
- `modules/integrated_dcf_handler.py`：智能集成處理器
- `modules/debug_tools.py`：除錯與開發者工具
- `modules/i18n.py`：多語訊息管理

## 🚀 快速開始

### 系統需求
- Python 3.11+
- Windows 10/11 (已測試)
- 4GB+ RAM (建議 8GB+)
- 網路連接 (用於 FinMind API)

### 安裝與執行
1. **克隆專案**
   ```bash
   git clone https://github.com/你的用戶名/jojo_trading.git
   cd jojo_trading
   ```

2. **安裝依賴**
   ```bash
   pip install -r requirements.txt
   ```

3. **啟動應用程式**
   ```bash
   streamlit run app.py
   ```

4. **訪問系統**
   - 開啟瀏覽器訪問: `http://localhost:8501`

## 📊 DCF與成長股邏輯

### DCF 估值模型
- **自由現金流模型**：基於營運現金流減去資本支出
- **EPS 成長模型**：基於每股盈餘成長預測
- **多階段成長**：支援高成長期與穩定期不同成長率
- **動態折現率**：Phase 1 新增 CAPM 模型自動計算

### Phase 1 增強邏輯
- **數據質量評分**：綜合評估財務數據的一致性、完整性、準確性
- **智能方法選擇**：高品質數據使用增強型 DCF，低品質數據自動備援
- **風險量化分析**：蒙地卡羅模擬提供估值分布與風險指標
- **情景分析**：樂觀、基準、悲觀三種情景下的估值對比

## 🎛️ Phase 1 控制面板

### 增強功能設定
- **啟用增強型 DCF**：切換按鈕控制 Phase 1 功能
- **數據質量閾值**：0.5-0.9 可調整品質要求
- **情景分析**：啟用多情景並行計算
- **蒙地卡羅模擬**：模擬次數與波動率調整
- **異常檢測**：檢測敏感度即時調整

### 參數建議
#### 保守型投資者
- 數據質量閾值: 0.8
- 蒙地卡羅模擬: 2000+ 次
- 關注悲觀情景結果

#### 積極型投資者  
- 數據質量閾值: 0.6
- 重視樂觀情景分析
- 接受較高波動率設定

- **DCF估值流程**：
  1. 先過濾成長股（如CAGR>10%、ROE提升等）。
  2. 取得FCFE現金流、成長率、折現率（無風險利率+風險溢酬）。
  3. 支援多階段成長率與敏感度分析。
  4. 非成長股可用其他估值法（如股息折現、PE/PB等）。

- **參數來源**：
  - 無風險利率：台灣10年期公債殖利率（央行/債券資訊網API）。
  - 通膨率：主計總處CPI年增率（政府開放資料）。
  - 永續成長率：GDP、產業平均或通膨率。

## 使用說明

1. 啟動 `app.py`，於側邊欄選擇語言、設定除錯選項。
2. 選擇產業或個股，設定篩選條件與估值參數。
3. 點擊「開始篩選股票」，系統自動抓取資料、過濾成長股、進行DCF估值。
4. 結果可於主畫面檢視並下載報表。
5. 除錯模式可清除快取、調整API延遲、啟用模擬數據。
