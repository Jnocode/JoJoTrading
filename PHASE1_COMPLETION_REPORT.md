## 📊 核心功能特色

### 1. 數據質量驗證
- **多維度評分**: 一致性、完整性、準確性綜合評估
- **智能閾值**: 可配置質量標準 (預設 0.7)
- **異常檢測**: 自動識別數據異常與不一致性
- **即時反饋**: UI 即時顯示數據質量狀態

### 2. 增強型 DCF 計算
- **動態折現率**: CAPM 模型自動計算風險調整折現率
- **情景分析**: 樂觀/基準/悲觀三種情景並行計算
- **蒙地卡羅模擬**: 1000次模擬生成估值分布與風險指標
- **敏感性分析**: 關鍵參數變化對估值影響評估

### 3. 智能方法選擇
- **質量驅動**: 根據數據質量自動選擇最適計算方法
- **無縫切換**: 高質量數據使用增強功能，低質量數據備援
- **透明處理**: UI 清楚顯示當前使用的計算方法
- **錯誤恢復**: 完善的異常處理與錯誤恢復機制
