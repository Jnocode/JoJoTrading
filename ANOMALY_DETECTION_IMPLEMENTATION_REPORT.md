## 📊 技術規格

### 檢測參數
- **預設閾值**: 1.5x 歷史平均 FCF_EPS
- **可調範圍**: 1.1x - 3.0x
- **歷史期數**: 4期（可配置）
- **最少期數**: 2期以上才進行檢測

### 檢測模式
- **嚴格模式**: 歷史數據充足(≥3期)且超過歷史最大值20%
- **寬鬆模式**: 歷史數據不足(<3期)但超過加嚴閾值1.3倍
- **極端模式**: 超過2倍閾值直接排除

### 數據品質評估
- **變異係數(CV)**: < 0.5 視為穩定，< 1.0 為不穩定，≥ 1.0 為極不穩定
- **品質評分**: 1.0 / (1.0 + CV)，滿分1.0，最低0.1
- **期數要求**: 至少2期有效數據才進行檢測
