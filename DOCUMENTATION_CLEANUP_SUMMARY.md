## 專案架構總覽

### 狀態機流程
```
CONFIG_LOAD → UI_INIT → INDUSTRY_PROCESS → DATA_FETCH → 
VALUATION → FILTERING → RESULTS_DISPLAY → EXPORT
```

### 核心技術棧
- **前端框架**: Streamlit
- **狀態管理**: 自訂狀態機模式
- **資料來源**: FinMind API + 台證所 OpenAPI
- **估值模型**: DCF (現金流量折現模型)
- **成長分析**: 多指標成長股篩選算法

### 模組職責分工
- `app.py`: 使用者介面與應用程式入口
- `jojo_state_machine.py`: 業務流程狀態機管理
- `data_handler.py`: 資料獲取、處理與快取
- `growth_analyzer.py`: 成長股分析與評估
- `i18n.py`: 多語言支援
- `debug_tools.py`: 開發工具與除錯輔助
- `xbrl_etl/`: XBRL 財務報表數據處理

## 使用指南

### 環境需求
- Python 3.8+
- Streamlit 1.28+
- 有效的 FinMind API 憑證
- 穩定網路連線

### 快速開始
1. 設定環境變數 (FinMind API 憑證)
2. 啟動 Streamlit 應用: `streamlit run app.py`
3. 選擇產業或輸入個股代碼
4. 調整 DCF 估值參數
5. 設定成長股篩選條件
6. 執行分析並查看結果

### 開發除錯
- 使用 `modules.debug_tools` 的快取清理功能
- 啟用模擬數據模式進行測試
- 調整 API 延遲以符合限制
